# Project Statement

RouteSense is a web-based maritime trajectory visualization project focused on how people perceive and interpret movement patterns on maps.

Many trajectory visualization tools emphasize data display, but RouteSense is interested in the interpretive layer: how visual choices such as color, contrast, path thickness, temporal segmentation, and anomaly highlighting affect a user's ability to notice unusual vessel behavior.

The project will begin with a simple JavaScript and ArcGIS/web mapping prototype. Later phases may include AIS data processing, vessel route filtering, basic anomaly indicators, and perception-aware interface experiments.

This project supports my preparation for thesis-level MSc Computer Science work by connecting practical software development with research interests in HCI, visual interpretation, perception, and representation.
